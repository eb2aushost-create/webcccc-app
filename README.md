# WEBCCCC Android App (WebView)

هذا مشروع Android Studio جاهز لتطبيق موقع https://www.webcccc.com داخل تطبيق.

## المميزات
- WebView داخل التطبيق + سحب للتحديث
- فتح الروابط الخارجية خارج التطبيق (غير webcccc.com)
- دعم تحميل الملفات (DownloadManager)
- دعم رفع الملفات من داخل الموقع (File chooser)
- زر الرجوع يرجع داخل الصفحات

## بناء APK
1) افتح المشروع في Android Studio
2) Build > Build Bundle(s)/APK(s) > Build APK(s)
3) ستجد APK في:
app/build/outputs/apk/debug/app-debug.apk

للنسخة النهائية:
Build > Generate Signed Bundle / APK
